
[[[[[[[[[[[[[ git 명령어 ]]]]]]]]]]]]]

>> 상태변경 확인 : git status
>> 히스토리 확인: git log 
>> git log 나가기 : q (혹은 :q)
>> 원격지에서 가지고오기 : git pull
>> 원격지에 올리기 : git push
>> 스테이지에 올리기 : git add 폴더 or 파일
>> 커밋하기 : git commit -m"메세지"
>> 브랜치확인 : git branch
>> 특정노드로 이동하기 : git checkout 브랜치 or commit id

[[[[[[[[[[[[[ 일반명령어 ]]]]]]]]]]]]]

>> 위치 옮기기 : cd 폴더명 
>> 한칸 뒤로가기 : cd ..
